# Imagens públicas do CRM

Use esta pasta para imagens que o frontend precisa servir direto.

## Pastas
- `owners/` : fotos de proprietários ou diretores
- `logos/` : logos e símbolos do CRM

## Sugestão de nomes
- `owners/owner-main.jpg`
- `logos/logo-crm.png`

## Como referenciar no Vue
Como essa pasta está em `public`, use caminhos começando com `/assets/...`.
Exemplos:
- `/assets/images/owners/owner-main.jpg`
- `/assets/images/logos/logo-crm.png`
